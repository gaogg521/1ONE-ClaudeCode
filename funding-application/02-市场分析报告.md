## 四、竞争格局分析

### 4.1 细分市场竞品对标

#### 对标 1：单一 Agent 平台（Cursor、Copilot）

```
产品：Cursor / GitHub Copilot

  ✓ 代码编写体验优化
  ✓ 集成度深
  ✓ 品牌认可度高

劣势：
  ✗ 只针对开发场景
  ✗ 不支持其他 Agent 切换
  ✗ 定时任务、自动化不行

vs 1ONE：
  → 1ONE 是"多 Agent 编排平台"，不是 IDE
  → 用户不会因为 Cursor 强而放弃 1ONE（使用场景不同）
  → 实际上 1ONE 可以集成 Cursor
```

```
产品：Make（原 Integromat）/ Zapier
定位：无代码工作流自动化
优势：
  ✓ 工作流成熟
  ✓ 集成数千个应用
  ✓ 社区活跃

劣势：
  ✗ 学习曲线陡峭（非技术用户上手难）
  ✗ 价格高（Zapier ¥300-1000/月）
  ✗ 云端存储，不适合敏感数据
  ✗ 不专注 AI，AI 功能不足

vs 1ONE：
  → 1ONE 是"AI-first 工作流平台"
  → 定价 10 倍便宜（¥9.99-299/月）
  → 本地化，数据隐私更好
  → 目标用户：创意工作者，不是 IT 运维

竞争策略：
  → 不与 Make 正面竞争（企业级自动化）
  → 聚焦"创意工作者 AI 自动化"细分市场
```

#### 对标 3：开源工作流平台（n8n、Airflow）

```
产品：n8n / Apache Airflow
定位：开源工作流引擎
优势：
  ✓ 完全开源
  ✓ 自托管，隐私好
  ✓ 社区强

  ✗ 部署复杂（Docker、Kubernetes）
  ✗ AI 功能微弱

  → 可与 n8n 社区合作

```

```
  - 数据在本地，模型在云端

  - 小模型本地运行（Ollama、LLaMA）

  - 混合部署（本地 + 云端 + 边缘）

  ✓ 建立"本地优先"的平台哲学
  ✓ 支持多部署模式（本地/云/混合）

```

```
  ├─ MAU：从 500 → 5000（第 3 个月）

  ├─ 首日留存：50%+
  ├─ 7 日留存：30%+
  ├─ 新用户 7 天内创建第一个工作流比例：20%+
  └─ 平均首次使用时间：15 分钟以内

  ├─ 30 日留存率：15-20%（业界标准 10-15%）
  ├─ 90 日留存率：8-10%
  ├─ 月度流失率：5-8%
  └─ 活跃用户平均使用频次：周均 3-5 次

付费（Monetization）
  ├─ 免费到付费转化率：3-5%
  ├─ 平均订阅生命周期：12-18 个月
  ├─ 月度 Churn Rate（企业版）：3-5%
  └─ 年度续费率：85%+

推荐（Referral）
  ├─ Net Promoter Score (NPS)：50+（业界优秀水准）
  ├─ 用户推荐意愿：40%+
  ├─ 平均每用户转介用户数（K 因子）：0.3-0.5
  └─ 转介到达率：20-30%
```
