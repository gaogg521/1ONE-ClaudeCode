## 📋 填充内容方法（任选一种）

### 方法 1️⃣：快速复制粘贴（推荐⭐⭐⭐）

最简单快速的方法：

1. 打开上方任一飞书链接
2. 在右侧本地 IDE 中打开对应的 `.md` 文件
3. 全选复制 (Ctrl+A / Cmd+A)
4. 在飞书文档中粘贴（Ctrl+V / Cmd+V）
5. 飞书会自动识别 Markdown 格式并美化排版 ✨

**所需时间**：约 2-3 分钟 / 文档

---

### 方法 2️⃣：一键导入（需要权限）

如果你是飞书工作区的管理员，可以使用导入功能：

1. 在飞书文档中，点击左上角 "..." 菜单
2. 选择 "导入" → "Markdown"
3. 上传本地 `.md` 文件
4. 等待飞书自动处理

**所需时间**：约 30 秒 / 文档
**前置条件**：需要飞书管理员权限

---

### 方法 3️⃣：使用飞书官方工具

下载飞书官方提供的 `markdown-to-docx` 转换工具（高级选项）。

**参考文档**：https://open.feishu.cn/document/ukTMukTMukTM/uUDN04SN0QjL1QDN

## ✨ 飞书 Markdown 支持的格式

飞书会自动识别并美化以下 Markdown 元素：

✅ 标题（# ## ###）
✅ 加粗（**text**）
✅ 斜体（*text*）
✅ 链接（[text](url)）
✅ 代码块（```code```）
✅ 列表（- 和 1.）
✅ 表格（| col | col |）
✅ 分隔线（---）
✅ 引用（> quote）
